Política de Privacidad para Pura Chan

Datos que recopilamos: El bot no almacena datos personales. Solo procesa IDs de Discord de servidores, canales y usuarios de forma temporal para ejecutar sus comandos.

Uso de la información: Los datos se utilizan únicamente para el correcto funcionamiento de las características del bot dentro de los servidores de Discord. No se comparten ni se venden datos a terceros.
