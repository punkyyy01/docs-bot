# docs-bot

Documentación oficial de **Purgatory Bot**, un bot de Discord orientado al entretenimiento.

## ¿Qué es Purgatory Bot?

Un bot que genera memes, reproduce música, y crea respuestas automáticamente usando cadenas de Markov. Diseñado para servidores que buscan diversión y caos controlado.

## Funcionalidades principales

- `/momo` — Generación de memes
- `/generar` — Texto automático por cadenas de Markov
- `/imitar` — Imita el estilo de escritura de un usuario
- `/corpus_wipe` — Limpia el historial de mensajes del bot
- `/corpus_ignorar` — Excluye canales del aprendizaje
- Reproducción de música desde YouTube y SoundCloud
